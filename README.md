# 🔬📖 X-ray GPT

Generate kindle-like x-ray for e-books with LLM

## 🚀 Usage

```shell
pip install xraygpt
python -m xraygpt [epub_file]
```

## LLM Support

Current this tool only support OpenAI and Azure OpenAI by setting environment variables.

## 🧑‍💻 Dev Setup
```shell
pip install -e '.[dev]'
```

### 🎩 Static analysis
```shell
make format
```
